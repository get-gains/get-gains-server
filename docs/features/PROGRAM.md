# Program Management

## Overview

**Purpose:**  
The Program Management feature allows coaches to create training programs and assign routines to them.

**Scope:**  
This document covers program creation, routine assignment, and related request validation.

**Dependencies:**  
- Express.js
- Zod (request validation)
- Prisma (Program, Routine models)

**Entry Points:**  
- `src/routes/program.routes.ts`
- `src/controllers/program.controller.ts`
- `src/schemas/program.schema.ts`
